## PlantUML и Visual Paradigm

**Instant Reverse** не импортирует PlantUML (только Java, C#, XML и т.д.).

1. Сгенерируйте PNG: https://www.plantuml.com/plantuml/uml/ → вставить `.puml` → Submit  
2. В VP: подложка через **Diagram → Import Active Diagram Image…** или рисуйте вручную по инструкции в `INSTRUCTION-VP-*.md`
